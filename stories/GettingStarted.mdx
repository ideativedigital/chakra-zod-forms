# Getting Started

Build forms from Zod schemas.

`zod-forms` provides `createZodForm(schema)` to generate typed form building blocks.

## Quick start

```tsx
import { z } from 'zod'
import { createZodForm } from 'zod-forms'

const schema = z.object({
  name: z.string().min(1),
  email: z.string().email(),
})

const { Form, Field, SubmitButton } = createZodForm(schema)
```
